# Figure-Making Guidelines

Please see the following links for figure-making guidelines:
- [GitHub repository](https://github.com/PennLINC/figure_formatting/tree/main?tab=readme-ov-file)
    - Contains Python and R scripts for formatting plots while enforcing a specified plot size in mm, as well as consistent font type and font size
    - Contains example usage scripts
    - Contains templates for multi-panel figures in Indesign and Illustrator
- [GitHub pages walk-through](https://pennlinc.github.io/figure_formatting/)
    - Contains guidelines for PennLINC-style figure tips and tricks, as well as a walthrough on how to use the above figure formatting scripts and templates to generate your manuscript figures
